# Retail Sales Analysis Project

## Domain
Retail

## Objective
Analyze retail sales data to identify trends, patterns and business insights.

## Dataset
Sample Superstore Dataset

## Analysis Performed
- Data Exploration
- Statistical Summary
- Sales Analysis
- Profit Analysis
- Correlation Analysis
- Data Visualization

## Visualizations
- Sales by Category
- Profit by Category
- Top States by Sales
- Sales vs Profit
- Correlation Heatmap

## Key Findings
- Technology generated the highest sales.
- Discounts negatively affected profit.
- Certain states produced higher revenue.
- Sales and Profit were positively related.

## Conclusion
The project provided valuable retail business insights through data analysis and visualization.
